# Dual-Kernel Graph Community Contrastive Learning

Tested with Python 3.7, PyTorch 1.12.1, and PyTorch Geometric 2.3.1, dgl 1.0.2.

```bash
pip install pandas
pip install scikit_learn
pip install numpy
pip install scipy
pip install einops
pip install ogb
pip install pyyaml
pip install googledrivedownloader
pip install networkx
pip install gdown
pip install matplotlib
```

For the question dataset, you can go into ./code, and then run the following command:

```
python main-question.py questions
```

For the other datasets, you can go into ./code, and then run the following command:

```
python main cora
```
